# var、let 及 const 区别

什么是提升？什么是暂时性死区？var、let 及 const 区别？

虽然变量还没有被声明，但是我们却可以使用这个未被声明的变量，这种情况就叫做提升，并且提升的是声明。

console.log(a) // undefined
var a = 1

var a = 10
var a
console.log(a) // 10

函数会被提升，并且优先于变量提升。

console.log(a) // ƒ a() {}
function a() {}

使用 var 声明的变量会被提升到作用域的顶部。
